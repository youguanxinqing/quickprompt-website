Let your AI manage
your prompts.

Install this skill and your agent can search, save and edit prompts in Quick Prompt just by talking to it. Everything stays on your Mac — no network, no keys.

⬇ Download the skill

macOS · requires Quick Prompt for Mac and a skill-capable agent

🔎

Ask what you have

“Do I already have a prompt about code comments?” — the agent reads your library and answers.

💾

Save on the fly

“Save this as a prompt.” It opens the editor in Quick Prompt for you to glance at and keep.

✏️

Tweak existing ones

“Add a {{language}} blank to that comment prompt.” Updates land as pending review, never silently.

🔒

Local & safe

Reads a local file, writes only through the app. No deletes, no network, no API keys.

Install

  1. Download and unzip quickprompt-skill.zip.
  2. Drop the quickprompt/ folder into your agent's skills directory: 
    mkdir -p ~/.claude/skills
cp -R quickprompt ~/.claude/skills/
  3. Restart your agent. Then just talk to it — “存进 Quick Prompt”, “我有没有关于 X 的 prompt”, “改一下那条 prompt”.
